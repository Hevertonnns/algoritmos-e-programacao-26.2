📘 UC – Algoritmos e Programação

Este repositório reúne os materiais, exercícios e códigos desenvolvidos ao longo das aulas da Unidade Curricular de Algoritmos e Programação.

O objetivo é registrar o conteúdo trabalhado em sala, organizar exemplos práticos e disponibilizar listas de exercícios para acompanhamento da evolução dos estudantes.

📂 Organização do Repositório

O conteúdo está organizado por aula.

Cada aula possui sua própria pasta contendo:

📄 Códigos desenvolvidos em sala

📝 Exercícios propostos

📘 Materiais complementares (quando houver)

Exemplo de estrutura:

📦 algoritmos-programacao

 ┣ 📂 aula01
 
 ┃ ┣ 📄 exemplos.py
 
 ┃ ┣ 📄 lista_exercicios.md
 
 ┣ 📂 aula02
 
 ┃ ┣ 📄 operadores.py
 
 ┃ ┣ 📄 lista_exercicios.md
 
 ┣ 📄 README.md
 
🎯 Objetivo da Disciplina


Desenvolver o raciocínio lógico e a capacidade de resolução de problemas por meio da programação, utilizando Python como linguagem base.

Ao longo das aulas, são abordados temas como:

* Variáveis e tipos de dados

* Entrada e saída de dados

* Operadores aritméticos e relacionais

* Estruturas condicionais

* Estruturas de repetição

* Funções

* Estruturas de dados básicas

🛠 Linguagem Utilizada

* Python 3

Este repositório tem finalidade exclusivamente educacional e serve como apoio às atividades da disciplina.
